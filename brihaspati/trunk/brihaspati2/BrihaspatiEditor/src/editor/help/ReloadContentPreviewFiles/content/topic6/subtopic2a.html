File Menu
The File menu contains all the standard file action options you would expect to find in any piece of software, as well as some specific actions releavant to RELOAD Editor.

New
The New Menu item (File, New >) provides a choice for the user of creating a new IMS Metadata file, a new IMS 1.2.2 Content Package or a new SCORM 1.2 Content Package.

Open
The Open Menu item (File, Open >) provides access to an Open Dialog box to open existing IMS or SCORM Content Packages or Metadata files.

Import Resources
The Import Resources ... Menu item (File, Import Resources ... ) allows the user to import new resources into a content package from anywhere else on their computer/system. The menu item brings up a file chooser dialogue box and enables the user to navigate to the content of their choice. Once content is identified, the file(s) are copied over to the working directory of the current content package. Whole directories can be added in this way. If the user chooses to import an individual file, they may also choose to check the checkbox 'Include dependent Files' which will ensure that any associated images or stylesheets are also copied into the Content Package working directory.

Save
The Save Menu item (File, Save) allows the user to save the information for their current Content Package.

Save As ...
The Save As ... Menu item (File, Save as ...) allows the user to save the information for their current Content Package with a new name. The main reason you might want to do this is if you want to create a package with the same structure as your current one, but wish to have different Unique Identifiers for 'identifier' attributes.

Zip Content Package ...
The Zip Content Package ... Menu item (File, Zip Content Package ...) allows the user to save their current Content Package.

Save Content Package Preview ...
The Save Content Package Preview ... Menu item (File, Save Content Package Preview ...) allows the user to save a set of files corresponding to the Reload Preview of their Content Package. These files themselves do not constitute a Content Package.

Recent Files
The Recent Files Menu item (File, Recent Files >) provides access to four recently edited Content Packages.

Exit
The Exit Menu item (File, Exit) quits the RELOAD Editor. You will be asked to save any unsaved work.
