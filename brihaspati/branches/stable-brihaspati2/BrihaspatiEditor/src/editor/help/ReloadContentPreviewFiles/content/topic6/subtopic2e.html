Window Menu
The window menu provides a list of currently open files (content packages) and provides options for how these open windows are arranged on the screen.

Cascade
The Cascade Menu Item (Window, Cascade) allows the user to organise open windows into a cascade pattern for easy on screen navigation.

Tile Horizontally
The Tile Horizontally Menu Item (Window, Tile Horizontally) allows the user to tile windows down the screen for easy comparison.

Tile Vertically
The Tile Vertically Menu Item (Window, Tile Vertically) allows the user to tile windows across the screen for easy comparison.

Open Windows
Below the menu separator on the Window Menu, a list of all open Content Package windows is presented.
