1. Application Overview

 

PMS is used for managing all type projects of any organization.

 

1.1. Objective

 

Project Management System (PMS) is web based project tracking application. It allows creating portal, organizations and defining roles, projects, tasks, scheduling the dates and assigning the task to the user. Its basic objective is to view the status of project and activity status of the user.

The portal concept is used for achieving the multi-Institutional concepts.

 

1.2. Features

 

• Web Based Solution

 

Being a web based application; it is possible for a user to remain in sync with the progress of tasks for projects. This allows one to assign, update and monitor the tasks when away from office.

 

• Graphical representations of project wise status

 

Unlike other project tracking application, PMS gives the graphical representations of the progress of the project.

 

• Creating project team

 

The application allows selecting the team for each project defined in the system.

 

• Assigning the tasks to the selected member

 

Once the member is added for a project, it is possible to assign the tasks to members of the project with scheduled dates.

 

• Controlled access

 

The whole system is worked on the basis of roles. On the basis of roles, it can be defined which user should have access to which screen this is known as authorities of the user.
